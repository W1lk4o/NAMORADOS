# 📖 Tutorial — Publicar no GitHub Pages

## O que você vai precisar
- Conta gratuita no GitHub (github.com)
- Os 4 arquivos deste projeto: `index.html`, `style.css`, `script.js`, `photos.js`

---

## Passo a passo

### 1. Criar conta no GitHub
Acesse **github.com** e crie uma conta gratuita (se ainda não tiver).

---

### 2. Criar um repositório
1. Clique no botão **"New"** (ou **"+"** no canto superior direito > New repository)
2. Preencha:
   - **Repository name:** `nossa-historia` (ou qualquer nome sem espaços)
   - Deixe marcado como **Public**
   - Clique em **"Create repository"**

---

### 3. Fazer upload dos arquivos
1. Na página do repositório, clique em **"uploading an existing file"**
2. Arraste os 4 arquivos de uma vez:
   - `index.html`
   - `style.css`
   - `script.js`
   - `photos.js`
3. Role a página para baixo e clique em **"Commit changes"**

---

### 4. Ativar o GitHub Pages
1. Clique em **Settings** (aba no topo do repositório)
2. No menu lateral esquerdo, clique em **"Pages"**
3. Em **"Source"**, selecione:
   - Branch: **main**
   - Pasta: **/ (root)**
4. Clique em **Save**

---

### 5. Aguardar a publicação
O GitHub vai processar em 1–3 minutos.  
Seu site estará disponível em:

```
https://SEU_USUARIO.github.io/nossa-historia/
```

> Exemplo: `https://wilker123.github.io/nossa-historia/`

---

### 6. Gerar o QR Code
Com o link em mãos, acesse um destes sites gratuitos para gerar o QR:

- **qr-code-generator.com**
- **qrcode-monkey.com** (permite personalizar cor)

Cole o link do seu site, gere o QR e imprima ou mostre na tela para Ana Aressa escanear. 🥰

---

## Dica: como atualizar as fotos
Se quiser trocar ou adicionar fotos no futuro:
1. Gere um novo `photos.js` com as imagens atualizadas
2. No repositório, clique no arquivo `photos.js`
3. Clique no ícone de lápis (editar) ou arraste o novo arquivo
4. Confirme as alterações com **"Commit changes"**

---

## Problemas comuns

| Problema | Solução |
|----------|---------|
| Site não abre | Aguarde mais alguns minutos após ativar o Pages |
| Fotos não aparecem | Certifique-se de que `photos.js` foi enviado |
| Música não toca | Alguns celulares bloqueiam autoplay — o usuário precisa clicar em "Começar" |
| QR Code não funciona | Verifique se o link do Pages está correto |
